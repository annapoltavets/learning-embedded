# Проєкт 6 — Керування LED через Wi-Fi (ESP32-S3)

ESP32-S3 запускається як точка доступу (AP), розгортає HTTP-сервер і надає
веб-сторінку для увімкнення/вимкнення світлодіода та регулювання його яскравості
через PWM.

## Схема підключення

```
 ┌─────────────────────────────────────────────────┐
 │              ESP32-S3 DevKit-C1                  │
 │                                                  │
 │  3V3 ○                                ○ GND      │
 │  EN  ○                                ○ IO43     │
 │  IO4 ○                                ○ IO44     │
 │  IO5 ○                                ○ IO1      │
 │  IO6 ○                                ○ IO2      │
 │  IO7 ○                                ○ IO42     │
 │  IO15○                                ○ IO41     │
 │  IO16○                                ○ IO40     │
 │  IO17○                                ○ IO39     │
 │  IO18○                                ○ IO38     │
 │  IO8 ○                                ○ IO37     │
 │  IO3 ○                                ○ IO36     │
 │  IO46○                                ○ IO35     │
 │  IO9 ○                                ○ IO0      │
 │  IO10○                                ○ IO45     │
 │  IO11○                                ○ IO48     │
 │  IO12○                                ○ IO47     │
 │  IO13○                                ○ IO21     │
 │  IO14○                                ○ IO20     │
 │  5V  ○                                ○ IO19     │
 │  GND ○─────────────────────────────────────────┐ │
 └───┬─────────────────────────────────────────────┘ │
     │ IO16 (PWM)                                     │
     │                                                │
     ▼                                                │
  ┌──┴──┐   220 Ω   ┌─────┐                          │
  │LEDC ├──[═══]────┤  A  │►  K  ├───────────────────┘
  │ch 0 │           │    LED     │         GND
  └─────┘           └─────┘
```

| ESP32-S3 | Компонент         | Призначення                        |
|----------|-------------------|------------------------------------|
| GPIO 16  | 220 Ω → анод LED  | PWM-вихід (LEDC канал 0)           |
| GPIO 0   | вбудована кнопка BOOT | керування яскравістю фізично   |
| GND      | катод LED         | Спільна земля                      |

> Резистор 220 Ω обмежує струм: при 3.3 В і падінні 2 В на LED → I = (3.3−2)/220 ≈ 6 мА.
> Кнопка BOOT вже є на платі й підключена до GPIO 0 із підтяжкою до 3.3 В — зовнішніх компонентів не потрібно.

## Як PWM регулює яскравість

LEDC (LED Control) — апаратний PWM-периферій ESP32-S3. Він автоматично
генерує на піні GPIO 16 імпульсний сигнал із частотою **5 кГц**.
Яскравість визначається **шириною імпульсу (duty cycle)** — часткою
часу, коли пін HIGH за один період.

```
  brightness = 255 (100%)        Яскравість максимальна
  ┌───────────────────────────┐
  │                           │
──┘                           └──

  brightness = 128 (50%)         Яскравість ~50%
  ┌─────────┐         ┌─────────┐
  │         │         │         │
──┘         └─────────┘         └──

  brightness = 64 (25%)          Яскравість ~25%
  ┌────┐             ┌────┐
  │    │             │    │
──┘    └─────────────┘    └─────────

  brightness = 0 (0%)            LED вимкнено
──────────────────────────────────
```

Людське око не встигає побачити мигання на 5 кГц,
тому сприймає середню яскравість — чим довший імпульс,
тим яскравіше світить LED.

```
ledcWrite(pwmChannel, brightness);  // 0 = вимк, 255 = max
```

Кнопки `+` / `−` на веб-сторінці та фізична кнопка BOOT змінюють `brightness`
з кроком 25 (~10%), після чого `ledcWrite` оновлює duty cycle на льоту.

## Маршрути HTTP

| URL         | Дія                            |
|-------------|--------------------------------|
| `/`         | Головна сторінка               |
| `/on`       | Увімкнути LED                  |
| `/off`      | Вимкнути LED                   |
| `/brighter` | Збільшити яскравість на 10%    |
| `/dimmer`   | Зменшити яскравість на 10%     |

## Вбудована кнопка BOOT (GPIO 0)

Кнопка BOOT вже розміщена на платі — підключати нічого не треба.
Вона підтягнута до HIGH через внутрішній резистор (`INPUT_PULLUP`),
тому натискання дає LOW (active-low).

**Поведінка:**
- Кожне натискання збільшує `brightness` на 25.
- Після максимуму (255) значення повертається до 0 і цикл починається знову.
- Якщо LED був вимкнений — автоматично вмикається.
- Антидребезг програмний на `millis()`, затримка 30 мс.

```
Послідовність натискань:
  0 → 25 → 50 → 75 → 100 → ... → 250 → 275(→0) → 25 → ...
  (яскравість в одиницях 0–255)
```

## PWM (LEDC)

| Параметр    | Значення      |
|-------------|---------------|
| Канал LEDC  | 0             |
| Пін         | GPIO 16       |
| Частота     | 5 000 Гц      |
| Розрядність | 8 біт (0–255) |
| Крок зміни  | 25 (≈ 10%)    |

## Як використовувати

1. Прошити ESP32-S3 через PlatformIO.
2. Підключити смартфон до Wi-Fi мережі `ESP32_LED_Control` (пароль `password123`).
3. Відкрити браузер і перейти за адресою `http://192.168.4.1`.
4. Керувати LED кнопками на сторінці.
